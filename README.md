## 项目简介

基于 Spring Boot + Thymeleaf + PostgreSQL 的学生考勤管理系统。

### 主要功能

- 学生管理（增删改查）
- 考勤打卡（自动判断迟到）
- 考勤记录查询与筛选
- 批量导入 Excel 考勤数据
- 考勤统计与图表展示
- 多角色权限控制（学生/教师/管理员）

### 技术栈

- Spring Boot 3.2.5
- Spring Security
- Thymeleaf
- PostgreSQL
- Bootstrap 5
- Apache POI
